Detailed Description

SONAR Audio Engine - Shared Library Interface.

Author
Mojibake Development Team
Version
1.0.0

This header defines the API for the SONAR audio engine shared library. Provides functions for audio initialization, playback, WAV generation, and file output capabilities for data sonification.

◆ audio_sample_node_t

typedef struct audio_sample_node audio_sample_node_t

Audio sample structure for linked list storage.

Represents a single audio sample with frequency, amplitude, duration, and source byte information. Must match the structure in SONAR extension.
