Mojibake Extension Development Guide

How to Create Your Own Extensions

Mojibake uses a modular architecture that makes it easy to add new file analysis capabilities.

Step 1: Create Header File

Create a new .h file in mojibake/modules/default/:

#ifndef MBX_YOURMODULE_H
#define MBX_YOURMODULE_H
#include <stdbool.h>
#include "../../mojibake.h"
// Your module function
bool mbx_yourmodule(mojibake_target_t *target, unsigned int index, void *arg);
#endif

Step 2: Implement Your Module

Create a corresponding .c file:

#include "mbx_yourmodule.h"
#include <stdio.h>
bool mbx_yourmodule(mojibake_target_t *target, unsigned int index, void *arg)
{
if (target == NULL || target->block == NULL || index >= target->partition_count)
return false;
unsigned char *partition = MOJIBAKE_BLOCK_OFFSET(target, index);
// Your analysis logic here
printf("=== Partition %d Your Analysis ===\n", index);
for (int i = 0; i < target->partition_size; i++) {
unsigned char byte = partition[i];
// Process each byte...
}
return true;
}

Step 3: Add to Main Program

  1. Include your header in the main program
  2. Add a new case in the module selection logic
  3. Recompile with your new module

Available Data

  • target->size: Total file size
  • target->partition_count: Number of partitions
  • target->partition_size: Size of each partition
  • partition: Pointer to current partition data
  • index: Current partition index

Example Extensions Ideas

  • Entropy Calculator: Measure randomness/compression potential
  • File Type Detector: Identify file formats by magic numbers
  • String Extractor: Find readable strings in binary files
  • Checksum Calculator: Compute MD5, SHA1, CRC32
  • Pattern Finder: Search for specific byte patterns
  • Image Analyzer: Detect image dimensions and format
  • Compression Analyzer: Identify compressed sections

Compilation

gcc -o mojibake_extended mojibake_extended.c mojibake/mojibake.c \
mojibake/modules/default/mbx_default.c \
mojibake/modules/default/mbx_yourmodule.c \
# ... other modules

The modular design makes Mojibake infinitely extensible for any file analysis need!
