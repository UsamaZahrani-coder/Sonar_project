Detailed Description

Mojibake SONAR Edition - Main Program.

Author
Mojibake Development Team
Version
1.0.0a

Main program for the Mojibake SONAR Edition, providing audio-based file analysis through SONAR (data-to-audio) and dSONAR (audio-to-data) conversion capabilities.
